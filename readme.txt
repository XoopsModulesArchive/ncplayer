
What's Ncplayer ?

Ncplayer is a xoops module. The goal is that you can listen music when you browse article.  
It provide some convenience method to add or delete music address. You can click the music name 
to play or select multi music , next to click multiplay. Enjoy you !

How to install ?

1. install as a normal module
2. add below script to your theme.html
<head>
<script src="<{$xoops_url}>/modules/ncplayer/include/jquery-1.2.1.js" type="text/javascript"></script> 
</head>
3. that's all.
